## ✨ Features

- **Dark Theme Design** - Eye-friendly dark mode (#0b0c10 background) with white text for optimal readability
- **Smooth Animations** - Page transitions, hover effects, scroll reveals, and glitch animations
- **Fully Responsive** - Works seamlessly on desktop, tablet, and mobile devices
- **Multi-Page Layout** - Home, About, Experience, Projects, and Contact pages
- **Research Showcase** - Dedicated section for academic publications and research projects
- **Project Filtering** - Categorized project display with filter tabs (Vision, LLM, Real-time, Research)
- **GitHub Integration** - Stats and links to open source contributions
- **Interactive Timeline** - Visual work experience timeline with hover effects
- **Contact Form** - Functional contact form with validation
- **Page Transitions** - Smooth fade transitions between pages

## 📄 Pages Overview

### 🏠 Home Page (`index.html`)
- Hero section with AI/Computer Vision focus
- Specialization cards (Vision Transformers, LLMs, Real-time Streaming, Computer Vision)
- Featured projects showcase
- Skills cloud with interactive tags
- Publications section
- Testimonials from collaborators
- Current work and interests

### 👤 About Page (`about.html`)
- Personal bio and philosophy
- Education timeline
- Research interests grid
- Awards and recognition
- Community involvement (speaking, workshops, open source)

### 💼 Experience Page (`experience.html`)
- Professional stats (years, companies, models deployed)
- Modern timeline cards with:
  - Company logos
  - Role descriptions
  - Key achievements with metrics
  - Tech stack tags
  - Impact badges
- Key projects at work
- Speaking engagements and workshops

### 🚀 Projects Page (`projects.html`)
- Project stats (total projects, GitHub stars, API calls, research papers)
- Filter tabs (All, Computer Vision, LLMs & NLP, Real-time, Research)

  - Real-Time Student Residence Identity Verification System
  - DermAssist Skin Cancer Diagnostic System
  - Document Scanner AI
  - Code Assistant

  1. Cybersecurity: Personal Systems Threat Mitigation
  2. Vision Transformers for Identity Verification
  3. DermAssist: Hybrid Vision Transformer for Skin Lesion Diagnosis
  4. EdgeViT++: Efficient Vision Transformers for Edge Devices

- GitHub activity stats

### 📞 Contact Page (`contact.html`)
